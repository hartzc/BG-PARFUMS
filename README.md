# Projeto de Site de Perfumes

Este projeto foi desenvolvido por nossa equipe de três criadores apaixonados, dedicados a oferecer uma experiência online moderna e elegante para nosso cliente do setor de perfumes.  

Nosso objetivo foi criar um site que refletisse a sofisticação e a singularidade de cada fragrância, garantindo ao mesmo tempo uma navegação fluida e intuitiva para os usuários.  

### Equipe
- [hartzc ](https://github.com/hartzc)
- [migzai ](https://github.com/migzai-g)
- [ogxrtur ](https://github.com/ogxrtur) 

### Funcionalidades
- Design responsivo para todos os dispositivos  
- Vitrine elegante dos produtos  
- Navegação simples e experiência de visualização prática  
- Performance otimizada e estrutura de código limpa  

Temos orgulho de apresentar este projeto como resultado de trabalho em equipe, criatividade e atenção aos detalhes.
