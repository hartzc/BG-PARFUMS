BG Parfums - Landing Page

Este projeto é uma landing page moderna para uma loja de perfumes, desenvolvida com foco em uma experiência premium, design responsivo e alta performance.

O projeto ainda está em desenvolvimento e novas funcionalidades serão adicionadas gradualmente, incluindo páginas secundárias, catálogo de produtos, detalhes dos perfumes, melhorias na interface e otimizações gerais.
